# Universal Downloader

A premium all-in-one media downloader for images, videos, reels, and GIFs.

## Supported Platforms

- Pinterest
- Twitter/X
- Instagram
- Facebook
- And more...

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```
